# knn-project


Program se spouští:

python main.py --model_type 0 --model_dir baseline_data

model_type 0 odpovídá baseline, model_type 1 odpovídá ECAPA a model_type 2 hybridnímu WavML + ECAPA

model_dir zase složce pro uložení modelu/načtení modelu

Datová sada i všechny potřebné věci se stáhnou při prvním spuštění
